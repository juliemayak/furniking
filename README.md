# Vue.js starter by frōntensō

## Project setup

```
yarn install
```

or

```
npm install
```

### Compiles and hot-reloads for development

```
yarn start
```

or

```
npm start
```

### Compiles and minifies for production

```
yarn build
```

or

```
npm run build
```

### Lints and fixes files

```
yarn lint
```

or

```
npm run lint
```
